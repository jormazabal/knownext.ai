# KnowNext.ai 0.7.0

KnowNext.ai 0.7.0 introduces the FastAPI-mediated AI documentation assistant and project RAG foundation.

## Changes

- Added a fixed `IA` workspace tab for every active project with persisted project conversation, day grouping, assistant/user messages, and file-operation events.
- Added OpenAI provider configuration in app settings with local credential storage, API key preview/status, and action permission toggles.
- Added AI interactions through FastAPI only. React never calls OpenAI directly and receives structured operation results from the local API.
- Added AI document edits that apply to the active editor buffer and remain `Cambios sin guardar` until the user saves normally.
- Added the right-aligned AI response bubble for informational answers, including close, `Abrir en IA`, and dissolve transitions.
- Added AI-assisted folder/document creation when permissions allow it, plus delete requests that always require explicit UI confirmation.
- Added opt-in RAG with OpenAI vector stores, per-project local manifests, incremental Markdown indexing, local SQLite exact search, and richer index status.
- Updated backend/frontend architecture docs and the manual acceptance checklist for AI, permissions, conversation, RAG, and deletion flows.
