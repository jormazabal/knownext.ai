# KnowNext.ai 0.3.0

KnowNext.ai 0.3.0 establishes monolithic application versioning across the desktop frontend, Tauri shell, Rust package metadata, and FastAPI backend.

## Highlights

- Shared `VERSION` release source for the whole project.
- Version consistency check with `pnpm version:check`.
- Release gate with `pnpm release:check`.
- Application version shown in the account/configuration layer.
- Backend `/health` response includes the application version.

## Validation

- Run `pnpm release:check`.
- Complete `docs/development/manual-test-checklist.md` before publishing a packaged desktop release.

## Known Limitations

- Git version history is still mocked.
- AI behavior is still mocked through the FastAPI contract.
- Packaged Tauri sidecar management for the FastAPI backend is not yet complete.
