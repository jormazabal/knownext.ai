# KnowNext.ai 0.13.0

Release date: 2026-05-15

## Summary

KnowNext.ai 0.13.0 adds the first realtime audio transcription workflow and polishes image asset handling in the document workspace. The microphone control now supports transcribing into the AI prompt or dictating into the active Markdown document, while the backend owns the OpenAI Realtime connection and keeps provider details out of React components.

## Product Changes

- Added a split microphone control in the AI prompt with direct start/stop behavior and a compact options menu.
- Added transcription destination choices for `Transcribir al prompt` and `Dictar en documento`.
- Added language selection from automatic mode plus favorite languages configured in application settings.
- Added configurable transcription model settings, defaulting to the realtime Whisper-compatible OpenAI flow.
- Added document dictation insertion through the active Milkdown cursor selection.
- Removed dictation command handling from this scope; punctuation commands remain a future product decision.
- Improved global tooltip behavior so clicking menu/action controls hides their tooltip before overlays appear.
- Improved image viewer metadata with file type, size, pixel dimensions, color depth, update time, and indexing state.
- Added image viewer zoom controls from 10% to 200% plus fit-to-window.
- Simplified image usage references into compact clickable Markdown document rows.

## Technical Changes

- Added FastAPI transcription routing and a backend transcription service for OpenAI Realtime WebSocket session mediation.
- Added frontend transcription API helpers, audio capture, PCM worklet processing, and React state management for realtime transcription sessions.
- Updated backend configuration schemas for transcription model and language preferences.
- Extended image asset metadata contracts with dimensions and color depth.
- Hardened Markdown image reference parsing for filenames with spaces.
- Aligned frontend backend discovery with expected app version and compatible local API profiles.
