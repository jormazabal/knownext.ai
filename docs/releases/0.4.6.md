# KnowNext.ai 0.4.6

KnowNext.ai 0.4.6 restores top-edge height resizing for the custom Windows title bar.

## Changes

- Added a native top-edge resize hit area to the frameless Tauri title bar.
- Preserved title-bar dragging, double-click maximize/restore, and minimize/maximize/close buttons.

## Validation

- Run `pnpm release:check`.
- Launch the installed Windows app.
- Confirm dragging the top edge reduces and expands the window height.
- Confirm dragging the title area still moves the window.
- Confirm double-clicking the title area still toggles maximize/restore.
- Confirm minimize, maximize/restore, and close still work.
