# KnowNext.ai 0.6.0

KnowNext.ai 0.6.0 hardens the application as a real data workspace instead of a mock-backed prototype.

## Highlights

- The app no longer falls back to sample projects, documents, users, version history, or AI answers during normal runtime.
- Startup now shows a clean loading layer before resolving to the real workspace, first-project state, or an API/runtime error.
- Empty installations show `Añadir primer proyecto` with explanatory copy instead of a project dropdown.
- Project creation is organized into responsive tabs/steps so the flow remains usable on smaller windows.
- The Tauri window minimum size is lower, while desktop split panels remain width-resizable and persisted.
