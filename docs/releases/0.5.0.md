# KnowNext.ai 0.5.0

KnowNext.ai 0.5.0 packages the GitHub/versioning foundation and the compact workspace improvements built after the 0.4.x updater releases.

## Changes

- Added GitHub login scaffolding, repository listing, and FastAPI-mediated auth contracts.
- Added local Git and GitHub API versioning service contracts with project capability checks.
- Added project sync actions, version creation, remote conflict handling, and history availability messaging.
- Added document draft recovery, orphan draft handling, and disk-change checks for open documents.
- Improved the compact editor workspace with responsive navigation, lazy editor loading, stable toolbar state, and native Tauri window controls.
- Expanded frontend and backend tests around project modes, draft behavior, document sync, version creation, and GitHub conflict paths.

## Validation

- Run `pnpm release:check`.
- Build Tauri with updater signing enabled through the release workflow.
- Confirm the published release contains NSIS, MSI, both updater signatures, and `latest.json`.
- Confirm `latest.json` resolves to `0.5.0`.
- Confirm `windows-x86_64.url` in `latest.json` points to `KnowNext.ai_0.5.0_x64_en-US.msi`.
- Confirm the README installer URL resolves to `KnowNext.ai_0.5.0_x64-setup.exe`.
