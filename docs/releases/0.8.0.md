# KnowNext.ai 0.8.0

KnowNext.ai 0.8.0 improves AI document workflows, makes prompt execution more cost-aware, and adds direct drag-and-drop organization in the project tree.

## Changes

- Added `Rapido` and `Razonar` prompt modes so simple tasks can stay direct while complex tasks can use structured reasoning with selectable depth.
- Added structured AI pending intents that preserve the target document across follow-up messages and expose explicit actions for applying, cancelling, or allowing web research.
- Separated conversational AI answers from document content changes so replies are not written into the active document unless the response includes a validated document change.
- Added selected-text focus context in the prompt input, plus active document folder context for document creation and project operations.
- Added move and duplicate action support tied to creation permissions in the AI configuration and execution flow.
- Added AI usage surfaces and backend accounting foundations grouped by model, interactions, tokens, and estimated cost.
- Redesigned the AI conversation tab, prompt input, response bubbles, and pending action controls with compact rounded controls and improved scrolling.
- Added drag-and-drop movement for documents and folders in the project tree, including root drop support, auto-expansion, and validated backend moves.
- Updated the document status bar so the save button only appears when the active document has pending changes, with document metrics aligned separately from history/status details.
