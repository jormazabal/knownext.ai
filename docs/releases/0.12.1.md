# KnowNext.ai 0.12.1

KnowNext.ai 0.12.1 polishes the professional workspace introduced in 0.12.0, focusing on desktop layout control, modal treatment, editor ergonomics, and dark-mode integration.

## Changes

- Added resizable integrated desktop panels so the document navigation and version history widths can be adjusted by dragging their separators.
- Added an integrated collapsed left rail for desktop mode, including a show-documents control and compact account/application menu access.
- Improved project selector alignment and document panel controls so the left panel adapts cleanly to its configured width.
- Added subtle background blur to modal overlays and improved the recoverable drafts dialog with clearer explanatory text, a stronger empty state, and standardized app buttons.
- Simplified Markdown block format labels in the editor toolbar and refined toolbar, tab, tooltip, and hover states in dark mode.
- Reworked the AI prompt controls so the microphone appears before the final send action, and the send action uses the selected primary color.
- Updated the document status bar to prioritize document type, word count, and character count, while hiding Git/history status when versioning is not active for the project.
- Reduced excessive editor empty space and limited the prompt fade to a short transition so short documents do not show unnecessary scrollbars.
