# KnowNext.ai 0.6.3

KnowNext.ai 0.6.3 hardens diagnostics after application settings introduced local trace logging.

## Highlights

- Expected local API connection failures from background settings and draft checks no longer show repeated user notices.
- Repeated frontend diagnostics are de-duplicated within a short interval to keep `knownext.log` readable.
- Tauri trace-log writes are serialized so concurrent errors cannot mix two JSON entries into one line.
- Trace entries written by Tauri now use stable Unix millisecond timestamps instead of Rust debug output.
