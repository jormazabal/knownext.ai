# KnowNext.ai 0.4.5

KnowNext.ai 0.4.5 changes the Windows update artifact used by the in-app updater.

## Changes

- `latest.json` now prefers the MSI artifact for Windows in-app updates.
- The NSIS setup `.exe` remains available as the manual installer from the README and GitHub Releases.
- Removed the custom NSIS install hook so the NSIS installer uses Tauri's default process detection and restart behavior.

## Validation

- Run `pnpm release:check`.
- Build Tauri with updater signing enabled.
- Confirm the published `latest.json` points to `KnowNext.ai_0.4.5_x64_en-US.msi` for `windows-x86_64`.
- From an installed `0.4.2`, trigger `Buscar actualizaciones` and confirm the app updates to `0.4.5`.
- Confirm the manual `KnowNext.ai_0.4.5_x64-setup.exe` installer remains downloadable.
