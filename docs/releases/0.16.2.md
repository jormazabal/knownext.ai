# KnowNext.ai 0.16.2

## Summary

KnowNext.ai 0.16.2 is a hotfix for importing larger batches of external files.

## Fixes

- Serializes backend Git operations per project so automatic external-change scans do not overlap with imports.
- Pauses the frontend external-change polling loop while an import is in progress.
- Uses non-locking Git status reads for passive scans.
- Retries short-lived Git `index.lock` contention before surfacing an error.
- Reduces scan work by reusing one Git status result when inferring new parent folders.

## Validation

- `pnpm version:check`
- `pnpm release:check`
- Backend regression test for transient `index.lock` retry.
- Backend external-change API flow using accented paths.

## Distribution

The GitHub release for `v0.16.2` must include the NSIS installer, MSI installer, both updater signatures, and `latest.json`. The published `latest.json` must resolve to version `0.16.2` and keep `windows-x86_64.url` pointing to the MSI artifact.
