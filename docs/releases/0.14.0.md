# KnowNext.ai 0.14.0

Release date: 2026-05-16

## Summary

KnowNext.ai 0.14.0 adds first-class AI image generation and completes the settings redesign needed to configure multimodel AI capabilities from a compact professional modal. Users can ask the assistant in natural language to create an image from the active document or selected text, save it as a normal project image asset, open it in an image tab, and optionally insert it into Milkdown as a rendered Markdown image.

## Product Changes

- Added prompt-driven AI image generation without command syntax or language-specific keyword matching.
- Added generated image assets to the project tree using the same image handling as uploaded project images.
- Added optional document insertion so generated images can be returned as relative Markdown references and rendered visually in Milkdown.
- Added configurable image-generation model, size, quality, output format, metadata, destination folder, and insertion behavior.
- Added separate permissions for generating images, creating image files, inserting images into documents, and using document context for image generation.
- Redesigned `Configuración de la app` around document-style tabs: Resumen, Interfaz, IA documental, Capacidades, and Sistema y diagnóstico.
- Added a configuration summary view with compact cards for interface, document AI, capabilities, system diagnostics, and selected AI models.
- Added a reusable AI model selector with recommendation tags, capacity/cost meters, compact rendering, and combobox-style option behavior.
- Redesigned the Interface and Capabilities settings screens for the narrower application modal width.
- Added a compact AI usage panel with monthly totals by capability and by model.

## Technical Changes

- Extended FastAPI AI interaction contracts with structured `imageGeneration` plans and generated image response metadata.
- Added backend execution for GPT Image generation through `openai_service`, with asset creation delegated to `asset_service`.
- Added usage accounting for document AI, image generation, image vision indexing, realtime transcription, and agentic tasks.
- Added fallback usage reconstruction from generated-image metadata so images created before image usage events still appear in monthly usage.
- Extended AI configuration schemas and migrations for image generation, vision, transcription, permissions, and agentic limits.
- Added frontend API/types for generated images, multimodel usage summaries, and the expanded settings surface.
- Updated architecture, product, and manual validation documentation for AI image generation and multimodel accounting.
