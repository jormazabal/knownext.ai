# KnowNext.ai 0.6.12

KnowNext.ai 0.6.12 improves the project editor so it matches the richer creation flows introduced in recent releases without exposing unsafe project-type changes.

## Changes

- The project editor now uses a maintenance-focused layout instead of a simplified creation form.
- Editable fields are limited to project identity, appearance, and the local folder reference when the project is not a GitHub-managed cache.
- Origin, storage mode, history mode, sync mode, and GitHub repository are shown as read-only technical configuration with explanatory copy.
- Saving an edited project preserves existing storage, versioning, sync, and GitHub metadata.
- Frontend tests now cover GitHub-backed project editing and verify that technical configuration remains read-only.
