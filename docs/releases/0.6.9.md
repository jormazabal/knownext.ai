# KnowNext.ai 0.6.9

KnowNext.ai 0.6.9 improves the first project setup experience and prepares local documentation folders for GitHub publication.

## Changes

- Reworked the project creation dialog into a clearer assistant with ordered scenarios, required-step validation, and a final review screen before creating the project.
- Split new local folder setup into parent location and folder name, with a visible final disk path so users can understand exactly what will be created.
- Added the "Create GitHub repo from local" flow for local folders, including repository owner, repository name, private/public visibility, remote repository creation, and local Git `origin` configuration.
- Added clearer disabled-state explanations for GitHub-only options.
- Persisted the desktop window position and size across application restarts.
