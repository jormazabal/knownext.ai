# KnowNext.ai 0.11.0

KnowNext.ai 0.11.0 adds first-class image management for documentation projects and refines the settings experience so image, IA, and indexing controls are easier to understand and operate.

## Changes

- Added image files to the project tree with image icons, project/folder import flows, and filtering between all files, Markdown documents, and images.
- Added image viewing inside the application, including image tabs alongside Markdown documents.
- Added Markdown image insertion from existing project images or from disk, storing newly uploaded images next to the active Markdown document.
- Added Markdown image rendering in the document view so referenced project images appear inline where they are used.
- Added reference handling for moving images and Markdown documents, including link rewrites and user-facing warnings where references could break.
- Added image context support in the AI prompt and backend context resolution so project images can be sent to OpenAI vision-capable models.
- Added configurable IA vision settings for model selection, detail level, maximum prompt images, maximum image size, image indexing, and visual-description storage.
- Redesigned the project tree toolbar and settings modal tabs for a more compact, operational desktop workflow.
- Improved the IA settings panel with compact OpenAI key handling, vision controls, action permissions, agentic limits, and RAG index status.
