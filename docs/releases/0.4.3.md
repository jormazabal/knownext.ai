# KnowNext.ai 0.4.3

KnowNext.ai 0.4.3 fixes two Windows packaging issues found in the installed app.

## Changes

- Release builds no longer open a separate terminal window when KnowNext.ai starts.
- The NSIS installer now closes a running `knownext-ai-desktop.exe` before replacing files during update installation.
- Passive/silent update installations relaunch KnowNext.ai after the installer finishes.

## Validation

- Run `pnpm release:check`.
- Install `0.4.2`, update to `0.4.3`, and confirm the installed app version changes.
- Confirm launching KnowNext.ai from the Start Menu does not open a terminal window.
- Confirm `latest.json` resolves to version `0.4.3`.
