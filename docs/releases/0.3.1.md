# KnowNext.ai 0.3.1

This release adds the first production updater path for the Windows desktop app.

## Added

- Signed Tauri updater support using public GitHub Releases.
- Manual update checks from the account menu.
- Silent startup update checks with failure-safe behavior when offline.
- User confirmation before update installation.
- Draft persistence before installation starts.
- GitHub Actions release workflow for Windows updater artifacts and `latest.json`.

## Validation

- Install `0.3.0`, publish `0.3.1`, and confirm the app detects, installs, and relaunches into the new version.
- Confirm local documents, projects, drafts, tabs, and configuration remain intact after update.
- Confirm offline checks and invalid signatures do not block app startup or normal editing.
