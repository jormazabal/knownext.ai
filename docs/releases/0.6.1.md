# KnowNext.ai 0.6.1

KnowNext.ai 0.6.1 adds application-level settings for appearance and diagnostics, with opt-in local trace logging for troubleshooting.

## Highlights

- Added `Configuración de la app` from the account menu.
- Added a settings modal with `Apariencia` and `Trazas` sections.
- Added persisted language and zoom settings for the local interface.
- Added an opt-in trace logging toggle.
- Added a dedicated local `logs/knownext.log` file for user-visible errors and runtime failures.
- Added an action to open the dedicated log folder from settings when trace logging is enabled.
