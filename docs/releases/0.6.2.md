# KnowNext.ai 0.6.2

KnowNext.ai 0.6.2 fixes application settings behavior in packaged builds when the local FastAPI backend is unavailable.

## Highlights

- The trace log folder is now resolved through Tauri runtime commands, so it no longer stays on `Preparando carpeta de logs`.
- The log folder action remains available from settings without requiring the local API.
- Appearance and diagnostics preferences are stored locally as a fallback when the API cannot persist `config.json`.
- The settings modal, account menu, and first-project state now update between Spanish and English when the language selector changes.
- The zoom slider keeps the standard arrow cursor instead of switching to a text cursor.
