# KnowNext.ai 0.5.1

KnowNext.ai 0.5.1 restores the release-notes experience for installed users updating from 0.5.0.

## Changes

- Added `Notas de release` to the account menu.
- Added a global readonly release notes tab rendered with Milkdown from the packaged `CHANGELOG.md`.
- Kept release notes outside project document trees, drafts, saves, history, and AI document context.
- Added config compatibility for installations that did not yet have release-note version state.

## Validation

- Run `pnpm release:check`.
- Confirm the published release contains NSIS, MSI, both updater signatures, and `latest.json`.
- Confirm `latest.json` resolves to `0.5.1`.
- Confirm `windows-x86_64.url` in `latest.json` points to `KnowNext.ai_0.5.1_x64_en-US.msi`.
- Confirm the README installer URL resolves to `KnowNext.ai_0.5.1_x64-setup.exe`.
