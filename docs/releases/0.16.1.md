# KnowNext.ai 0.16.1

## Summary

KnowNext.ai 0.16.1 is a hotfix for the external-change import flow introduced in 0.16.0.

## Fixes

- Correctly decodes Git quoted UTF-8 paths for pasted folders and files with accents or other non-ASCII characters.
- Prevents imports from sending mojibake paths such as `FacturaciÃ³n/Facturaciones.md` to Git.
- Keeps the external-change review drawer and import action aligned with the actual filesystem path, for example `Facturación/Facturaciones.md`.

## Validation

- `pnpm version:check`
- `pnpm release:check`
- Backend regression test for Git quoted UTF-8 path decoding.
- Backend external-change API flow using `Facturación/Facturaciones.md`.

## Distribution

The GitHub release for `v0.16.1` must include the NSIS installer, MSI installer, both updater signatures, and `latest.json`. The published `latest.json` must resolve to version `0.16.1` and keep `windows-x86_64.url` pointing to the MSI artifact.
