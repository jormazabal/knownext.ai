# KnowNext.ai 0.4.1

KnowNext.ai 0.4.1 is a documentation release that makes the public Windows installer easy to find from the repository README.

## Changes

- Added a README download section with the latest Windows installer link.
- Linked the public GitHub Releases page.
- Documented the signed updater manifest endpoint used by the desktop app.

## Install

- Latest Windows installer: https://github.com/jormazabal/knownext.ai/releases/latest/download/KnowNext.ai_0.4.1_x64-setup.exe
- Releases page: https://github.com/jormazabal/knownext.ai/releases

## Validation

- Run `pnpm release:check`.
- Confirm the release contains `KnowNext.ai_0.4.1_x64-setup.exe`, the matching `.sig` file, and `latest.json`.
- Confirm `latest.json` resolves to version `0.4.1`.
