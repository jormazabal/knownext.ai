# KnowNext.ai 0.4.0

KnowNext.ai 0.4.0 is the first open source release.

## Added

- Apache-2.0 project license.
- `NOTICE` and third-party dependency review notes.
- Contribution guide, security policy, code of conduct, issue templates, pull request template, and CODEOWNERS.
- Public GitHub repository metadata and branch controls for reviewed contributions.

## Notes

- The updater remains signed with the existing Tauri updater key.
- The public updater endpoint now resolves through GitHub Releases.
- This release supersedes `0.3.1` as the public baseline because the legal and contribution files are included in the tag.
