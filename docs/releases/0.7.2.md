# KnowNext.ai 0.7.2

KnowNext.ai 0.7.2 polishes the AI interaction surface and adds explicit model selection for OpenAI-backed document assistance.

## Changes

- Moved AI applied-change notices to floating prompt-area bubbles so they no longer appear inside the document editor flow.
- Removed conversation and undo actions from the applied-change notice; closing the notice is local to the prompt area, while document undo remains available through editor controls.
- Added a right-aligned waiting bubble while an AI prompt is being processed.
- Rounded the AI prompt, AI response bubbles, and conversation bubbles more strongly to match a modern chat-style interaction surface.
- Added an AI model selector in `Configuración de la app > IA` with compact model cards, intelligence/cost indicators, and the recommended `gpt-5.4-mini` default.
- Persisted the selected model in the frontend and FastAPI configuration contract, and passed it to OpenAI Responses API calls.
- Fixed browser-development API discovery so the web app can select a compatible `web-dev` backend when stale or incompatible local backends are running.
- Fixed the account menu click behavior and copyable backend diagnostics feedback in the Services settings panel.

## Verification

- Run `pnpm release:check`.
- Open `Configuración de la app > IA` and confirm the model selector shows `gpt-5.4-mini`, `gpt-5.4`, `gpt-5.5`, and `gpt-5.4-nano` with visible intelligence/cost indicators.
- Select `gpt-5.5`, confirm it is marked selected, then return to `gpt-5.4-mini`.
- Submit an AI prompt and confirm a waiting bubble appears above the prompt while the response is pending.
- Submit an AI document edit and confirm the applied-change notice appears as a floating, right-aligned prompt-area bubble with only the summary and a close button.
- Open `Ajustes > Servicios`, click `Copiar diagnóstico`, and confirm the button shows copied feedback.
- Confirm `latest.json` resolves to `0.7.2`.
- Confirm `windows-x86_64.url` in `latest.json` points to `KnowNext.ai_0.7.2_x64_en-US.msi`.
- Confirm the README installer URL resolves to `KnowNext.ai_0.7.2_x64-setup.exe`.
