# KnowNext.ai 0.4.4

KnowNext.ai 0.4.4 fixes the custom Windows frame behavior in the installed desktop app.

## Changes

- The minimize, maximize/restore, and close buttons now call the Tauri window APIs.
- The top title area can be dragged to move the application window.
- Double-clicking the title area toggles maximize/restore.
- Invisible resize handles on the edges and corners allow resizing the frameless window.
- Tauri window permissions now explicitly allow the required native window operations.

## Validation

- Run `pnpm release:check`.
- Launch the installed Windows app and confirm no separate terminal window opens.
- Confirm minimize, maximize/restore, and close all work from the custom title bar.
- Confirm dragging the top title area moves the app window.
- Confirm dragging each edge and corner resizes the app window.
- Install `0.4.3`, update to `0.4.4`, and confirm the app closes, updates, and relaunches.
