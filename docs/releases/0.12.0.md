# KnowNext.ai 0.12.0

KnowNext.ai 0.12.0 introduces global appearance personalization with a professional dark mode, configurable primary color, and theme-aware brand treatment.

## Changes

- Added `Configuración de la app > Apariencia` controls for Light, Dark, and system-synchronized theme modes.
- Added primary color personalization using the same color range available for projects.
- Applied the selected primary color across app navigation, active states, focus rings, settings previews, document tree selection, editor chrome, and prompt actions.
- Reworked dark mode into a neutral graphite palette so it combines cleanly with every configurable primary color instead of relying on a black or blue-tinted surface.
- Improved dark-mode hover states for document tabs, editor toolbar actions, tooltips, and the AI prompt input.
- Added a reusable dynamic KnowNext.ai brand mark so the title bar logo, startup mark, and empty-state watermark inherit the selected primary color.
- Persisted theme mode and primary color in the local app configuration, with backend defaults and compatibility handling for older config files.
- Updated product, frontend/backend architecture, and manual validation notes for the appearance customization system.
