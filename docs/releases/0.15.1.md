# KnowNext.ai 0.15.1

Release date: 2026-05-17

## Summary

KnowNext.ai 0.15.1 refines the document editing workspace with clearer AI prompt controls, explicit transcription target icons, and a more stable editor cursor experience. The release focuses on small interaction details that make the workspace feel more predictable while moving between the Markdown document and the AI prompt.

## Product Changes

- Unified the visual treatment of the controls embedded in the AI prompt input.
- Kept the prompt controls compact while aligning their background, border, hover, and disabled states.
- Added a prompt-target microphone icon for transcribing into the AI prompt.
- Added a document-target microphone icon for transcribing directly into the active Markdown document.
- Added the same transcription target icons to the target selection menu.
- Made the Markdown editor caret orange when the document has focus.
- Kept a fixed, visible orange caret at the last document insertion point when focus moves to another control such as the AI prompt.

## Technical Changes

- Added a ProseMirror plugin that tracks the editor selection and renders a persistent caret decoration when the editor loses focus.
- Added CSS rules that hide the persistent caret while ProseMirror has focus, leaving the native blinking caret in control.
- Reused the existing KnowNext.ai accent token for caret and control hover states.
- Preserved the existing Milkdown editor integration and single-column document surface.

## Validation

- `pnpm --filter @knownext/desktop build`
- Browser validation against a real FastAPI-backed project with a Markdown document and the AI prompt input.

## Updater Notes

The GitHub release for `v0.15.1` must include the NSIS installer, MSI installer, both updater signatures, and `latest.json`. The published `latest.json` must resolve to version `0.15.1` and keep `windows-x86_64.url` pointing to the MSI artifact.
