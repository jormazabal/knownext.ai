# KnowNext.ai 0.9.0

KnowNext.ai 0.9.0 focuses on editor reliability, AI edit behavior, and a more professional Markdown authoring surface.

## Changes

- Added bounded in-memory undo and redo state for open Markdown documents, with disabled default controls and independent activation for undo and redo availability.
- Recorded AI-applied document modifications in the same undo history as manual edits so users can revert AI changes from the editor controls.
- Aligned AI behavior with configured permissions: permitted edit operations execute directly, while unavailable operations explain which IA configuration controls the permission.
- Improved active-document AI execution to avoid unnecessary clarification for direct document-editing prompts.
- Hardened local backend health, restart, timeout, and compatibility diagnostics for development and installed application flows.
- Redesigned the Markdown toolbar with a block format dropdown for normal text and H1-H6, grouped inline formatting, structure actions, insertion actions, and history actions.
- Added extended underline support using Markdown-compatible HTML output, controlled by a new Appearance setting that is enabled by default.
- Added a visual table picker with hover selection and manual row/column fallback.
- Fixed top document chrome so title, tabs, and toolbar stay stable while only the document canvas scrolls vertically.
- Removed redundant document path text from AI modification bubbles when the path is already in the bubble title.
