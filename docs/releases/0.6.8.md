# KnowNext.ai 0.6.8

KnowNext.ai 0.6.8 fixes GitHub device authorization from the desktop login dialog.

## Changes

- GitHub `slow_down` responses now keep the device flow pending and increase the polling interval instead of surfacing a raw error.
- The login dialog now keeps checking authorization while it is waiting and disables duplicate manual checks during an active request.
- The GitHub verification page opens through Tauri's shell integration in the desktop app, with a normal browser fallback during web development.
- Added backend contract coverage for `slow_down` device-flow responses.
