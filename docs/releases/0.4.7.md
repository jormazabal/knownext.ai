# KnowNext.ai 0.4.7

KnowNext.ai 0.4.7 documents the release workflow so future updates remain installable and discoverable for users.

## Changes

- Added release-management rules to `AGENTS.md`.
- Added `docs/skills/release-management-skill.md` for future release agents.
- Documented the manual installer versus in-app updater artifact contract.
- Updated the README installer link for `0.4.7`.

## Validation

- Run `pnpm release:check`.
- Build Tauri with updater signing enabled.
- Confirm the published release contains NSIS, MSI, both updater signatures, and `latest.json`.
- Confirm `latest.json` resolves to `0.4.7`.
- Confirm `windows-x86_64.url` in `latest.json` points to `KnowNext.ai_0.4.7_x64_en-US.msi`.
- Confirm the README installer URL resolves to `KnowNext.ai_0.4.7_x64-setup.exe`.
